# Theme Provider
<P>
Wrap your application with ThemeProvider to manage the global theme (`'light'` or `'dark'`). 
Theme changes set the `data-theme` attribute on document root.
</P>
<ThemeProviderDisplay />
```jsx
import { ThemeProvider, ThemeContext, T } from '@kaynora/ui'
import { useContext } from 'react'

export default function () {
  return (
    <ThemeProvider>
      <App />
    </ThemeProvider>
  )
}
```
