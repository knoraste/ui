# Button
<P>
A polymorphic component that can render as a semantic `<button>` or `<a>` element depending 
on the props provided. This design exists to serve as a single source of 
truth for the appearance of button-like elements.
</P>
<ButtonDisplay />
```jsx
import { Button, T } from '@kaynora/ui'

export default function () {
  return (
    <Button>
      <T>Submit</T>
    </Button>
  )
}
```

## Examples

### Action
<P>
The `action` prop accepts either a `string` representing `href`, or a `function` representing 
`onClick`. Based on value type, renders a native `<a>` or `<button>` respectively.
</P>
<ActionsDisplay />
```jsx
import { Button, Modal, T } from '@kaynora/ui'

export default function () {
  return (
    <>
      <Modal
        isOpen={isModalOpen}
        onOpenChange={() => setIsModalOpen(false)}
      ><T>You're all caught up!</T></Modal>

      <div style={{
        display: 'flex',
        flexFlow: 'row wrap',
        alignItems: 'center',
        gap: '20px',
      }}>
        <Button action={() => setIsModalOpen(true)}><T>See notifications</T></Button>
        <Button action='#'><T>Scroll to top</T></Button>
      </div>
    </>
  )
}
```

### Surface
<P>
The `surface` prop determines the visual style of the button.
</P>
<SurfacesDisplay />
```jsx
import { Button, T } from '@kaynora/ui'

export default function () {
  return (
    <div style={{
      display: 'flex',
      flexFlow: 'row wrap',
      alignItems: 'center',
      gap: '20px',
    }}>
      <Button surface='fill'><T>Dashboard</T></Button>
      <Button surface='outline'><T>Dashboard</T></Button>
      <Button surface='hollow'><T>Dashboard</T></Button>
      <Button surface='text'><T>Dashboard</T></Button>
    </div>
  )
}
```

### Size
<P>
The `size` prop determines the amount of padding on the button.
</P>
<SizesDisplay />
```jsx
import { Button, T } from '@kaynora/ui'

export default function () {
  return (
    <div style={{
      display: 'flex',
      flexFlow: 'row wrap',
      alignItems: 'center',
      gap: '20px',
    }}>
      <Button size='s'><T>Create post</T></Button>
      <Button size='m'><T>Create post</T></Button>
      <Button size='l'><T>Create post</T></Button>
    </div>
  )
}
```

### Width
<P>
The `width` prop determines whether the button stretches to fit its container or shrinks 
to fit its content.
</P>
<WidthsDisplay />
```jsx
import { Button, T } from '@kaynora/ui'

export default function () {
  return (
    <div style={{
      width: '100%',
      display: 'flex',
      flexFlow: 'column',
      gap: '20px',
    }}>
      <Button width='full'><T>Submit</T></Button>
      <Button width='auto'><T>Submit</T></Button>
    </div>
  )
}
```
