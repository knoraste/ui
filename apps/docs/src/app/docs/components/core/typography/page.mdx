# Typography
<P>
A component that enforces visual consistency in text content with semi-decoupled 
semantics and styles. Contextual defaults minimize design overhead.
</P>
<TypographyDisplay />
```jsx
import { T } from '@kaynora/ui'

export default function () {
  return (
    <>
      <T type='h1'>Heading</T>
      <T>Paragraph text</T>
    </>
  )
}
```

## Examples

### Size
<P>
The `size` prop determines the font size of the text.
</P>
<SizesDisplay />
```jsx
import { T } from '@kaynora/ui'

export default function () {
  return (
    <T size='s'>Some small text</T>
  )
}
```
